## 🔧 功能介绍
- **CodeShare**：代码分享功能，帮助你分享代码片段。
- **WorkPlan**: 在这里存入你的计划，让你更好地管理时间。

## ⚙️ 如何启动项目

1. **确保你已经安装了 Go 和 Vue，并且配置好环境变量。**  
2. **克隆项目到本地：**
   
   ```bash
   git clone https://github.com/yourusername/Moment.git
   ```
3. **进入项目目录：**
   
   ```bash
   cd Moment
   cd backend
   ```
4. **安装项目依赖：**
   ```bash
   go mod tidy
   ```
5. **启动项目：**
   ```bash
   cd ..
   chmod +x start.sh   # 赋予脚本执行权限
   ./start.sh          # 启动项目
   ```

---

## 🧪 测试方法

1. 确保项目已成功启动。  
2. 打开浏览器，访问 `http://localhost:5173`测试即可。
